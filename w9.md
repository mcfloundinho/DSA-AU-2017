## 题目描述

给定一系列图，求在图中任意一个节点出发，经过至少两个不同节点后返回这一点所经历的最短环路长度，且要求环路至多经过每个节点一次。例如，从P1节点出发，需要环路经过P2,P3,...,Pk(k>2，且P1,P2,...,Pk中没有相同节点)后回到P1，求满足条件的最短环路的总长度。

## 输入

第一行为图的个数M（M<=100），接下来有M组输入。

每组中，第一行为图中节点数N（N<=100），接下来输入一个N*N的邻接矩阵。矩阵的第I行第J列代表从节点I到节点J道路的长度，其值大于等于0且小于10000，若I,J之间不存在道路，则为-1。

## 输出

总共M行输出，对应每组输入的图。对于每个输入的图，如果能找到这样一条最短环路的话，输出最短环路的最小值，如果找不到的话，输出-1。

## 样例输入

```
2
3
0 -1 1
-1 0 3
-1 1 0
4
0 3 2 3
5 0 3 4
6 3 0 1
7 2 3 0

```

## 样例输出

```
-1
6

```
